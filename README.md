# AtCoder
AtCoderの進捗を管理するためのもの
できるだけ毎日コードには触れて草を生やしたいので、GitHub Actionを使ってACを自動コミットするようにしたいと思っています。
### 3月14日 自動化
AtCoderでの精進記録をGitHub Actionsで自動同期しています。
AC（正解）だけでなく、WA（不正解）やCE（コンパイルエラー）もあえて記録することで、自分の思考の癖やデバッグの過程を振り返れるようにしています。
今日は円周率の日ですね。祝。
### 3月17日 バグ修正
毎日自動提出（Action）はされているはずなのに草が生えていないのに気づき原因究明。
結果メールアドレスをGitHubの自動Bot用のものでyamlに設定していたことが原因と分かった。じぶんのIDを含んだアドレスに変更。
